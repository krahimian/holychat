Holy Chat

A chat application for holy ship - build using express.js (node.js), redis, angular.js, and socket.io. The chat server runs on a static ip on the ships local area network. Clients (web or mobile) on the same network can connect to the server by either navigating to it's ip address or configuring it on the mobile (iOS & Android) app.
